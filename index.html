About This Project

KnowMe Genius is a mobile application focusing on psychological assessments and smart self-analysis. It provides a wide range of IQ tests, EQ tests, and personality quizzes to help users better understand themselves.

The project is currently under review for release on Google Play. In future updates, we plan to include more test categories and social features, such as friend challenges, result sharing, and AI-powered insights.
